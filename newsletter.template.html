AI Design

The Shift from Deterministic
to Probabilistic UI

As generative AI becomes integrated into core products, designers must move away from fixed layouts toward interfaces that can handle uncertainty and dynamic generation.

"The interface is no longer a fixed stage — it is a living negotiation between system intent and user context."

This shift demands new mental models: loading states that feel alive, error boundaries that maintain trust, and layouts that compose gracefully from generated content rather than dictating it.

从确定性 UI 向概率性 UI 的转变

随着生成式 AI 集成到核心产品中，设计师必须从固定布局转向能够处理不确定性和动态生成的界面。这一转变要求全新的心智模型：充满生命感的加载状态、维护用户信任的错误边界，以及能够优雅地适配生成内容的布局系统。
